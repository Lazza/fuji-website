Please carefully follow the installation procedure:

1. Download the latest version of Fuji [from GitHub](https://github.com/Lazza/Fuji)
2. Partition your destination drive using the **exFAT** file system
2. Set the volume label as `Fuji`
3. Download and copy the universal Fuji DMG in the drive
